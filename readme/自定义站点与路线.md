### 自定义站点与路线

您可以从云端获取路线，也可以自行添加站点和路线。

`站点`或者`路线`选项卡中罗列了所有的本地`站点`或`路线`。

要添加`站点`或`路线`，请点击屏幕下方的`加号按钮`。

#### 自定义站点

要添加站点，更便捷的操作是前往`设置`-`地图与定位`开启`单击地图添加站点`，然后回到`主控`。

在地图上点击要添加的站点的位置，Announcer会自动填入经纬度，您只需要输入其他信息即可完成添加。

#### 自定义路线

要自定义路线，必须先将路线途径的站点都添加到`自定义站点`中。

然后在`路线`选项卡点击下方的加号`+`按钮，从下列方式中选择一种来添加路线。

1. 在地图上选择站点录入（3.0 版本起）

    使用这种方式，地图会将所有的`本地站点`显示在地图上。您只需依次点击路线途径的站点，就可以完成路线的编辑。

    添加好站点后，点击左侧的对勾按钮`√`，完成编辑。此时可以继续编辑下行站点，也可以直接将上行站点反转，直接得到下行。

2. 通过站点ID录入

    每一个站点都有其唯一的数字ID，例如`解放桥[35]` `建干路口[170]`是不同的站点。

    值得注意的是，部分站点有上行和下行两个方向的站点。例如`乐群路口[62]` `乐群路口[238]`，它们虽然名称一样，但ID不一样，所以被看做两个不同的站点。

    一个站点可以停靠多条路线，所以不建议重复添加相同名称、相同方向的站点。

    例如`十字街(中心广场)[44]`就同时停靠了`10` `11` `100`等路线，而它们共用这一个站点。

    路线使用`站点ID序列`来表示上/下行的途径站点，每个站点用一个空格` `区分开来。

    例如`3路`上行的其中一段序列`159 91 239 46`就表示该路线上行依次停靠：

    `天鹅塘[159]` `桂林站[91]` `南门桥南[239]` `西城路口[46]` 这四个站点。

#### 数据库

Announcer采用`SQLite`数据库存储`站点`和`路线`，如有需要，您可以直接编辑数据库文件。

1. 前往`设置`-`数据与关于`，点击`备份站点与路线`。

2. 在`Announcer/Backups`文件夹中找到刚刚备份的`station.db`或`line.db`并提取，现在请按自己的方式编辑数据库文件。

3. 前往`设置`-`数据与关于`，点击`还原站点`或者`还原路线`，选择编辑好的文件。
